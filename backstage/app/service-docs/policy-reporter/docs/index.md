# Policy Reporter

Policy Reporter provides a Kubernetes-native dashboard for viewing and analyzing Kyverno policy reports, violations, and compliance status across your cluster.

## Overview

| Property | Value |
|----------|-------|
| **Namespace** | `policy-reporter` |
| **Type** | Policy Compliance Dashboard |
| **Default** | Disabled |
| **Config Key** | `flux_apps.policy-reporter` |
| **Deployment** | Flux HelmRelease |
| **Chart** | `policy-reporter` from kyverno.github.io |

## Official Documentation

- [Policy Reporter Docs](https://kyverno.github.io/policy-reporter/)
- [Kyverno Policy Reports](https://kyverno.io/docs/policy-reports/)
- [Policy Reporter UI](https://github.com/kyverno/policy-reporter-ui)

## Architecture

Policy Reporter watches for PolicyReport and ClusterPolicyReport CRDs generated by Kyverno and presents them through a web UI:

```
┌─────────────────────────────────┐
│        Policy Reporter UI       │
│         (Web Dashboard)         │
└──────────────┬──────────────────┘
               │
┌──────────────▼──────────────────┐
│       Policy Reporter API       │
│   (PolicyReport CRD Watcher)    │
└──────────────┬──────────────────┘
               │
┌──────────────▼──────────────────┐
│     Kyverno Policy Engine       │
│  (Generates PolicyReports)      │
└─────────────────────────────────┘
```

## Features

- **Kyverno Plugin**: Direct integration with Kyverno policy results
- **Dashboard UI**: Visual compliance status across namespaces
- **Policy Violations**: Drill down into individual policy failures
- **Cluster Reports**: View ClusterPolicyReport summaries

## Configuration

Policy Reporter is deployed via Flux HelmRelease with:

```yaml
values:
  kyvernoPlugin:
    enabled: true
  ui:
    enabled: true
    plugins:
      kyverno: true
```

## Enabling

```json
{
  "flux_apps": {
    "policy-reporter": true
  }
}
```

## Integration

Policy Reporter works alongside:
- **Kyverno**: Generates the PolicyReport CRDs that Policy Reporter displays
- **Kyverno Policies**: The actual policy definitions enforced by Kyverno
- **Grafana**: Policy Reporter metrics can be scraped by Prometheus
